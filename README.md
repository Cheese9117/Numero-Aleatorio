# Número Aleatorio

Programa de consola en **Python** que genera un número aleatorio dentro de un rango definido por el usuario, con validación de entrada. Proyecto de práctica.

## Uso

```bash
python NumeroAleatroio.py
```

El programa pide un mínimo y un máximo, valida que el rango sea correcto y muestra un número aleatorio entre ambos.

## Tecnología

Python 3 (módulo `random`)
